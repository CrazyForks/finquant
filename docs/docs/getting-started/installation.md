# 安装

## 环境要求

- Python 3.8+
- pandas >= 1.3.0
- numpy >= 1.20.0

## pip 安装

```bash
pip install finquant
```

## 源码安装

```bash
git clone https://github.com/meepo-quant/finquant.git
cd finquant
pip install -e .
```

## 可选依赖

```bash
# 安装所有可选依赖
pip install finquant[dev]

# 仅安装测试依赖
pip install pytest pytest-cov
```
